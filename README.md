## 1. 실행 방법

```bash
git clone https://github.com/blueprint-12/wanted-pre-onboarding-frontend.git

yarn && yarn start
```

## 2. 설치 패키지

스타일드 컴포넌트 (CSS in JS)

```bash
yarn add styled-components && yarn add -D @types/styled-components
```

axios (HTTP Client)

```bash
yarn add axios
```

react-router v6

```bash
yarn add react-router-dom && yarn add @types/react-router-dom

```
